## Como Usar

1. Extraia o arquivo `projeto_final_cores.zip`
2. Instale as dependências: `npm install`
3. Execute o projeto: `npm run dev`
4. Acesse em `localhost:3000`

A paleta de cores será aplicada automaticamente através das variáveis CSS definidas no arquivo `globals.css`.

## Notas Técnicas

- As cores foram definidas como variáveis CSS no escopo `:root` e `.dark`
- O sistema de temas está configurado para usar essas variáveis automaticamente
- Tailwind CSS está integrado e usará essas cores em todas as classes
- A paleta é totalmente responsiva e funciona em todos os dispositivos
