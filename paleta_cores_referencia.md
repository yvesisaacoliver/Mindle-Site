## Recomendações de Paleta para o Projeto

### Modo Dark (Principal)
- **Background**: #0B0E14 (fundo principal escuro)
- **Background Secundário**: #121824 (cards, seções)
- **Background Terciário**: #1A2332 (hover, estados)
- **Foreground/Texto**: #FFFFFF (texto principal)
- **Texto Secundário**: #B0B0B0 (texto menos importante)
- **Primary (Botões)**: #E31837 (vermelho vibrante)
- **Accent**: #25D366 (verde neon)
- **Border**: #1A448A (bordas azuis)
- **Muted**: #4A7DDA (elementos mutados)

### Paleta Complementar
- **Success**: #25D366 (verde)
- **Warning**: #FFA500 (laranja)
- **Destructive**: #E31837 (vermelho)
- **Info**: #4A7DDA (azul)

## Estratégia de Aplicação
1. Usar #0B0E14 como fundo principal
2. Usar #FFFFFF para textos principais
3. Usar #E31837 para botões e CTAs
4. Usar #25D366 para acentos e destaques
5. Usar variações de azul (#1A448A, #4A7DDA) para elementos secundários
6. Manter contraste alto para acessibilidade
